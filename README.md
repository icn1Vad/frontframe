# ProofSpace Frontend

ProofSpace 企业文件治理前端。项目保留 Next.js Pages Router，并在其上建立模块清单、持久 AppShell、类型化领域模型和可替换的数据端口。

```bash
npm ci
npm run dev
```

开发地址：`http://localhost:3000`

```bash
npm run typecheck
npm run lint
npm run test
npm run build
# 或一次运行全部检查
npm run check
```

前端结构和设计约束见 [`docs/FrontendArchitecture.md`](docs/FrontendArchitecture.md)。
